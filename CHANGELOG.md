# Changelog

Všetky dôležité zmeny v projekte **brloh-parser**.

## [0.1.0] - 2026-03-31
### Added
- prvá funkčná verzia brloh parsera
- zachovaný frontend, API kontrakt a event model podľa pikazard-parser
- baseline parser pre BRLOH.sk
- extrakcia názvu produktu
- extrakcia ceny
- extrakcia dostupnosti
- extrakcia URL produktu
- extrakcia obrázka do payloadu

### Changed
- scraping/parsing logika prispôsobená pre BRLOH.sk
- interný model ponechaný kompatibilný s ostatnými parser projektmi
